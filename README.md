# Maturitni_projekt
Maturitni projekt chytrý kvetinac

## Hardware:
- ESP 8266 WeMos D1 mini



## Funkce:
- Senzory na vlhkost, teplotu, vodu v zásobníku
- automatické zavlažování
- Displej
- webový server s grafy
- Předvolby pro jednotlivé květiny
